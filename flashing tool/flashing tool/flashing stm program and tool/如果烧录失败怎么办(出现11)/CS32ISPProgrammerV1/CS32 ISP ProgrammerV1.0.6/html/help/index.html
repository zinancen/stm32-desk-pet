1          CS32芯片生态链概述

芯片生态链是否丰富，会影响到整个产品的推广和应用，例如STM32的生态链就非常的丰富，他减轻了开发周期，提高开发效率和质量，并能完全释放开发者的创造能力。

CS32芯片的生态链参考STM32，建立起一个完整的生态系统，帮助用户快速获取帮助、接收最新技术动态、论坛互动、官方视频教程和Demo程序学习等。

3          IAR/Keil学习资源

官方提供IAR/Keil的CS32位芯片外设开发包，用户直接使用IAR/Keil开发环境进行开发，所以需要学习这些开发环境以及插件安装和使用。

3.1         外设包安装方法

首先在官网下载Pack，两种安装方式：

一、正常安装

          先正常安装xxxxx.pack，在keil界面上选择PackInstaller.exe进行安装。

二、手动安装

      直接解压xxxx.pack，将解压后的文件按照原目录结构复制到Keil\ARM\Pack\Keil\xxxxxxxx_DFP\a.b.c目录，其中xxxxxxx_DFP这个名字从解压后的文件Keil.xxxxx_DFP.pdsc里面找到，如：<name>STM32F0xx_DFP</name>，a.b.c为版本号，也从Keil.xxxxx_DFP.pdsc里面找到，如：<release version="1.5.0">。
